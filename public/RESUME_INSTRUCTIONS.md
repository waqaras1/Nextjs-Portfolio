# Resume Setup Instructions

To enable the resume download feature:

1. Create your resume as a PDF file
2. Name it `resume.pdf`
3. Place it in the `/public` folder
4. The download button will automatically work

The resume download button is located:
- On the home page (top right)
- On all internal pages via the navigation bar

Make sure your resume includes:
- Contact information
- Professional summary
- Work experience (including OneUpAI role)
- Education
- Technical skills
- Projects
- Certifications (if any)
